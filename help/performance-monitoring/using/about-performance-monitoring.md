# 关于性能监控 {#about-performance-monitoring}

控制面板提供了多项功能，可帮助您监测实例并确保实现最佳性能。

此 **[!UICONTROL 性能监控]** 使用控制面板主页中的信息卡，可监控Campaign实例的使用情况，如其数据库容量。 如需详细信息，请参阅[此部分](../../performance-monitoring/using/database-monitoring.md)。
