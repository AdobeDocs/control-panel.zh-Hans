# 最新版本 {#control-panel-releases}

此页面列出了控制面板的所有新增功能和改进。

## 2023年6月改进 {#june-2023}

**子域的SSL证书委派到Adobe**

您现在可以由Adobe管理子域的SSL证书。 如果您使用CNAME设置子域，则将自动生成并提供证书记录，以便在域托管解决方案中生成证书。 [了解详情](../subdomains-certificates/using/setting-up-new-subdomain.md)
